# 三模数 NTT

常数大、速度慢、精度高是它的特点。

在考虑三模数 NTT 之前先考虑一下中国剩余定理吧。

已知

{x≡x1(modm1)x≡x2(modm2)x≡x3(modm3)\begin{cases} x\equiv x_1(\bmod m_1)\\ x\equiv x_2(\bmod m_2)\\ x\equiv x_3(\bmod m_3)\\ \end{cases}⎩⎪⎪⎨⎪⎪⎧​x≡x1​(modm1​)x≡x2​(modm2​)x≡x3​(modm3​)​

求 xmodm1m2m3x\bmod m_1m_2m_3xmodm1​m2​m3​。

有

k1m1+x1=k2m2+x2k1≡x2−x1m1(modm2)x≡k1m1+x1(modm1m2)x4=(k1m1+x1)modm1m2k4m1m2+x4=k3m3+x3k4≡x3−x4m1m2(modm1m2m3)x≡k4m4+x4(modm1m2m3)\begin{aligned} &k_1m_1+x_1=k_2m_2+x_2\\ &k_1\equiv \frac{x_2-x_1}{m_1}(\bmod m_2)\\ &x\equiv k_1m_1+x_1(\bmod m_1m_2)\\ &x_4=(k_1m_1+x_1)\bmod m_1m_2\\ &k_4m_1m_2+x_4=k_3m_3+x_3\\ &k_4\equiv \frac{x_3-x_4}{m_1m_2}(\bmod m_1m_2m_3)\\ &x\equiv k_4m_4+x_4(\bmod m_1m_2m_3)\\ \end{aligned}​k1​m1​+x1​=k2​m2​+x2​k1​≡m1​x2​−x1​​(modm2​)x≡k1​m1​+x1​(modm1​m2​)x4​=(k1​m1​+x1​)modm1​m2​k4​m1​m2​+x4​=k3​m3​+x3​k4​≡m1​m2​x3​−x4​​(modm1​m2​m3​)x≡k4​m4​+x4​(modm1​m2​m3​)​

一点疑惑的解答（自言自语）：

因为 k1≡x2−x1m1(modm2)k_1\equiv \frac{x_2-x_1}{m_1}(\bmod m_2)k1​≡m1​x2​−x1​​(modm2​)，所以 k1=x2−x1m1+km2k_1=\frac{x_2-x_1}{m_1}+km_2k1​=m1​x2​−x1​​+km2​。又因为 k1m1≤m1m2k_1m_1\le m_1m_2k1​m1​≤m1​m2​，所以 k1≤m2k_1\le m_2k1​≤m2​。所以 k≥0k\ge 0k≥0，所以 k1k_1k1​ 最小为 x2−x1m1\frac{x_2-x_1}{m_1}m1​x2​−x1​​，即 x≡k1m1+x1(modm1m2)x\equiv k_1m_1+x_1(\bmod m_1m_2)\\x≡k1​m1​+x1​(modm1​m2​)。

进入正题：

所谓的三模数 NTT 指的是 以 998244353,1004535809,469762049998244353,1004535809,469762049998244353,1004535809,469762049 为模数（经典 NTT 模数，原根均为 333）分别进行 NTT，最后用上文的计算方式计算即可。

因为以上三个模数的乘积为很大，一般答案即使不取模也不会大于该数，所以上式的 k4m4+x4k_4m_4+x_4k4​m4​+x4​ 就是原答案，直接对题目给出的模数取模即可。

# 拆系数 FFT

常数小，速度快，精度低（longdouble⁡\operatorname{long double}longdouble 信仰跑）是它的特色。

如果直接对原数列进行 FFT 的话会炸精度的。考虑拆系数，即 Ai=J×Ai′+Ai′′A_i=J\times A'_i+A''_iAi​=J×Ai′​+Ai′′​（Ai′′<JA''_i< JAi′′​<J）。

那么：

F=A×B=(J×A′+A′′)×(J×B′+B′′)=J2A′B′+J(A′B′′+A′′B′)+A′′B′′\begin{aligned} F&=A\times B=(J\times A'+A'')\times(J\times B'+B'')\\ &=J^2A'B'+J(A'B''+A''B')+A''B''\\ \end{aligned}F​=A×B=(J×A′+A′′)×(J×B′+B′′)=J2A′B′+J(A′B′′+A′′B′)+A′′B′′​

如果直接计算的话需要四次 dft，三次 idft，和九次 ntt 的三模数 NTT 差距并不大。

考虑优化，然而 dft/idft 中有什么地方没有用到捏？虚部！考虑将 A′A'A′，A′′A''A′′，B′B'B′，B′′B''B′′ 合并在一起进行 dft。

设：

Pi=Ai′+Ai′′iQi=Ai′−Ai′′iEi=Bi′+Bi′′i\begin{aligned} P_i=A'_i+A''_ii\\ Q_i=A'_i-A''_ii\\ E_i=B'_i+B''_ii\\ \end{aligned}Pi​=Ai′​+Ai′′​iQi​=Ai′​−Ai′′​iEi​=Bi′​+Bi′′​i​

有：

Wi=(P×E)i=(Ai′Bi′−Ai′′Bi′′)+(Ai′Bi′′+Ai′′Bi′)iRi=(Q×E)i=(Ai′Bi′+Ai′′Bi′′)+(Ai′Bi′′−Ai′′Bi′)i\begin{aligned} &W_i=(P\times E)_i=(A'_iB'_i-A''_iB''_i)+(A'_iB''_i+A''_iB'_i)i\\ &R_i=(Q\times E)_i=(A'_iB'_i+A''_iB''_i)+(A'_iB''_i-A''_iB'_i)i\\ \end{aligned}​Wi​=(P×E)i​=(Ai′​Bi′​−Ai′′​Bi′′​)+(Ai′​Bi′′​+Ai′′​Bi′​)iRi​=(Q×E)i​=(Ai′​Bi′​+Ai′′​Bi′′​)+(Ai′​Bi′′​−Ai′′​Bi′​)i​

我们可以通过 WWW 和 RRR 的加减得到我们想要的系数。

Wi+Ri=2×(Ai′Bi′+A′Bi′′i)Ri−Wi=2×(Ai′′Bi′′−Ai′′Bi′i)Wi−Ri=2×(−Ai′′Bi′′+Ai′′Bi′i)\begin{aligned} &W_i+R_i=2\times(A'_iB'_i+A'B''_ii)\\ &R_i-W_i=2\times(A''_iB''_i-A''_iB'_ii)\\ &W_i-R_i=2\times(-A''_iB''_i+A''_iB'_ii)\\ \end{aligned}​Wi​+Ri​=2×(Ai′​Bi′​+A′Bi′′​i)Ri​−Wi​=2×(Ai′′​Bi′′​−Ai′′​Bi′​i)Wi​−Ri​=2×(−Ai′′​Bi′′​+Ai′′​Bi′​i)​
